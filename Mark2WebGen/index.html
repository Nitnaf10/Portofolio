Comment utiliser Mark2WebGen

Mark2WebGen est un outil simple pour générer du code HTML à partir de Markdown et de CSS. Voici comment l'utiliser :

1
Écrire en Markdown : Dans la zone de texte "Écrivez votre Markdown ici...", saisissez votre texte avec la syntaxe Markdown. Utilisez `#` pour les titres, `**gras**`, etc.

2
Ajouter du CSS : Personnalisez l'apparence du HTML en ajoutant vos propres styles dans la zone CSS.

3
Voir le résultat : Le rendu HTML est affiché à gauche. Le code source HTML apparaît à droite.

4
Appliquer des styles : Choisissez un style de citation (blockquote) ou une puce personnalisée pour vos listes.

5
Exporter : Vous pouvez télécharger le CSS, le HTML ou l’ensemble sous forme d’archive .zip.

Amusez-vous à créer et à styliser votre contenu web avec Mark2WebGen !
